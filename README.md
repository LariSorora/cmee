NPS CMEE – Acirp
Formulário de pesquisa de satisfação (NPS) das reuniões do Conselho da Mulher Empresária e Empreendedora (CMEE) da Acirp – São José do Rio Preto.
🔗 Acesso
O formulário está disponível em:
https://seu-usuario.github.io/nps-cmee/
📅 Como usar por reunião
Para cada reunião, gere um QR Code com a data (e tema opcional) na URL:
https://seu-usuario.github.io/nps-cmee/?data=2026-05-20
https://seu-usuario.github.io/nps-cmee/?data=2026-05-20&tema=Liderança
📊 Respostas
As respostas são coletadas automaticamente na planilha Google Sheets vinculada via Google Apps Script.
